# Smart To-Do List Application

A desktop-based task management application developed using Python, Tkinter, and SQLite.

## Features

* Add new tasks
* Set task priorities
* Search tasks
* Update existing tasks
* Mark tasks as completed
* Delete tasks
* Store tasks permanently using SQLite database

## Technologies Used

* Python
* Tkinter
* SQLite3

## Project Purpose

This application helps users organize daily tasks efficiently through a simple and user-friendly desktop interface.

## Future Enhancements

* Due dates and reminders
* Dark mode
* User authentication
* Data export to Excel
